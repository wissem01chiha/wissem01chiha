# Recommended Reading

## Reinforcement Learning

- **[Reinforcement Learning David Silver Course Slides](https://www.davidsilver.uk/teaching/)**  
  A comprehensive course on reinforcement learning by David Silver.
  
- **[DeepMind Reinforcement Learning Videos](https://www.youtube.com/watch?v=ISk80iLhdfU&list=PLqYmG7hTraZBKeNJ-JE_eyJHZ7XgBoAyb)**  
  A playlist of videos on reinforcement learning by DeepMind.

- **[Reinforcement Learning and Optimal Control](https://web.mit.edu/dimitrib/www/RLbook.html#:~:text=REINFORCEMENT)**  
  By Dimitri P. Bertsekas, MIT. This book covers the theory and practice of reinforcement learning.

## C++ Programming

- **[C++ Core Guidelines](https://isocpp.github.io/CppCoreGuidelines/CppCoreGuidelines#main)**  
  Guidelines to help write modern, safe, and efficient C++ code.

- **[Beautiful C++: 30 Core Guidelines for Writing Clean, Safe, and Fast Code](https://www.oreilly.com/library/view/beautiful-c-30/9780137647767/#:~:text=Expert%20C++)**  
  An O'Reilly book that provides insights into writing better C++ code.

- **GPU Parallel Program Development Using CUDA**  
  By Tolga Soyata. A detailed guide on developing parallel programs using CUDA.

- **Large-Scale C++ Software Design**  
  By John Lakos. A guide to designing scalable C++ software systems.

- **Parallel Programming in C with MPI and OpenMP**  
  By Michael Jay Quinn. This book introduces parallel programming concepts with practical examples.

- **Parallel Programming in OpenMP**  
  By Rohit Chandra, Ramesh Menon, Leo Dagum, et al. A comprehensive guide to programming with OpenMP.

- **Advanced C++: Master the Technique of Confidently Writing Robust C++ Code**  
  By Gazihan Alankus, Olena Lizina, Rakesh Mane, et al.

- **C++ Best Practices**  
  By Jason Turner. A collection of best practices for writing clean and maintainable C++ code.

- **Clean C++: Sustainable Software Development Patterns and Best Practices with C++ 17**  
  By Stephan Roth. A guide to applying clean code principles to C++.

## Julia Programming

- **Beginning Julia Programming for Engineers and Scientists**  
  By Sandeep Nagar. An introduction to Julia programming for scientific computing.

- **Data-Driven Science and Engineering: Machine Learning, Dynamical Systems, and Control**  
  By Steven L. Brunton, J. Nathan Kutz. A book that explores the intersection of data science and engineering.

- **Julia 1.0 Programming Complete Reference Guide**  
  By Ivo Balbaert and Adrian Salceanu. A complete reference guide for Julia programming.

- **Think Julia: How to Think Like a Computer Scientist**  
  By Allen Downey, Ben Lauwens. A beginner's guide to Julia programming.

## Computer Vision and Robotics

- **[Learning OpenCV: Computer Vision in C++ with the OpenCV Library](https://searchworks.stanford.edu/view/13213322)**  
  A comprehensive resource for learning computer vision with OpenCV.

- **Fundamentals of Computer Vision**  
  By Wesley E. Snyder, Hairong Qi. A foundational book on computer vision concepts.

- **Introductory Techniques for 3-D Computer Vision**  
  By Emanuele Trucco, Alessandro Verri. An introduction to 3D computer vision techniques.

- **Robot Vision**  
  By Berthold K.P. Horn. A classic book on robot vision.

- **Robotic Vision: Fundamental Algorithms in MATLAB®**  
  By Peter Corke. This book focuses on robotic vision algorithms implemented in MATLAB.

- **Three-Dimensional Computer Vision: A Geometric Viewpoint**  
  By Olivier Faugeras. A detailed exploration of 3D computer vision.

- **A First Course in Fuzzy and Neural Control**  
  By Hung T. Nguyen, Nadipuram R. Prasad, et al. An introduction to fuzzy and neural control systems.

## Selected Papers

- **Feedback Error Learning Neural Network Applied to a Scara Robot**  
  By Fernando Passold and Marcel0 Ricardo Stemmer.

- **Differential Neural Networks for Robust Nonlinear Control**  
  A study on robust nonlinear control using differential neural networks.

- **Identification, State Estimation and Trajectory Tracking**  
  By Alexander S. Poznyak, Edgar N. Sanchez, and Wen Yu.

- **Advanced Robotics: Redundancy and Optimization**  
  By Yoshihiko Nakamura, University of California, Santa Barbara.
